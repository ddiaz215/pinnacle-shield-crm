Instalación en Chrome
iMac · MacBook Pro · iPhone · iPad

Instalar CRM como App en Chrome · iMac

Guarda el archivo pinnacle-shield-crm-v11.html

Descárgalo y guárdalo en tu Mac. Recomendado: carpeta ~/Documents/Pinnacle Shield/

Abre el archivo en Chrome

Arrastra el archivo a Chrome o ve a File → Open File. La URL mostrará algo como file:///Users/tu-usuario/Documents/Pinnacle%20Shield/pinnacle-shield-crm-v11.html

Instalar como Progressive Web App (PWA)

En Chrome, haz clic en el ícono ⊕ que aparece en la barra de dirección (extremo derecho), o ve al menú ⋮ → Más herramientas → Crear acceso directo. Marca "Abrir como ventana".

Aparece en tu Dock y Launchpad

Chrome crea un ícono en el Dock de macOS. Haz clic secundario → Mantener en el Dock para que permanezca siempre visible.

Configura Supabase

Ve a la sección Configuración en el menú izquierdo, ingresa tu URL y API Key de Supabase, y presiona Guardar y conectar.

Instalar CRM como App en Chrome · MacBook Pro

Sincroniza el archivo vía iCloud Drive o carpeta compartida

Guarda pinnacle-shield-crm-v11.html en iCloud Drive → Pinnacle Shield para accederlo desde cualquier Mac.

Abre Chrome y navega al archivo

Arrastra el archivo a Chrome. La Supabase key está guardada en localStorage — solo configuras una vez por dispositivo.

Instalar como App

Menú Chrome: ⋮ → Guardar e compartir → Instalar página como aplicación. Marca "Abrir como ventana".

Agrega al Dock

La app aparece en Launchpad y Applications. Arrástrala al Dock para acceso inmediato. Los datos se sincronizan en tiempo real con Supabase.

Instalar CRM en iPhone via Chrome

⚠ En iOS, Chrome no permite instalar PWAs como Safari. Usaremos Chrome para acceder y guardar en pantalla de inicio vía Safari, o simplemente guardar un acceso directo.

Sube el archivo a iCloud Drive

En tu Mac, guarda pinnacle-shield-crm-v11.html en la carpeta iCloud Drive → Pinnacle Shield.

Opción A: Chrome Mobile

Abre la app Files en tu iPhone → iCloud Drive → Pinnacle Shield → toca el archivo HTML → se abre en Chrome. Selecciona el ícono de Compartir → "Agregar a pantalla de inicio".

Opción B: Abre en Chrome (recomendado para uso diario)

En Chrome iOS: barra de dirección → escribe la URL del archivo o usa un servidor local. Para acceso rápido: ⋮ → Agregar a pantalla de inicio.

Configura Supabase en el dispositivo

Una vez abierto, ve a Configuración y guarda tu Supabase URL y Key. Los datos se sincronizan automáticamente con tu cuenta cloud.

Instalar CRM en iPad via Chrome

Accede desde iCloud Drive

App Files → iCloud Drive → Pinnacle Shield → pinnacle-shield-crm-v11.html. Toca para abrir — iOS preguntará con qué app. Selecciona Chrome.

Configura Split View para productividad

En iPad con Chrome, puedes usar el CRM en Split View junto a WhatsApp, Email o Notas. Ideal para cotizar mientras chateas con clientes.

Agrega a pantalla de inicio

Con el archivo abierto en Chrome: toca el ícono Compartir → Agregar a pantalla de inicio. Se instala como ícono nativo en tu iPad.

Sincronización automática

El CRM detecta cuando hay conexión y sincroniza cambios con Supabase. Si estás offline, guarda localmente y sincroniza al reconectarte.
